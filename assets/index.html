HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin is the space around the entire element.
  • Padding is the space around the content inside an element.
  • CSS files are linked within the head element

Git

  • Git status: checks what branch we're currently on.
  • Git checkout-b branch name: creates a new branch and switches to it
  • Git add -A commits all files in the working branch.
  • Git commit -M Commits all changes
  • Git pull origin branch name syncs your local branch with the remote branch on GitHub
  • Git push origin feature/ pushes all your changes

Javascript

  • A function is followed by parentheses
  • A variable, "var" is a named container that stores data
  • Strings are always in quotation marks
  • Lines of Javascript end with a semicolon
  • JavaScript files are linked in the body element
  • Arrays are single variables that holds a group of data
  • You have to call a function for it to be executed
  • Control flow is the order in which a computer executes code in a script
